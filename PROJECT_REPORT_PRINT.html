Project Report

Title

Low-Cost 3D Surface Reconstruction Using Photometric Stereo

Abstract

This project presents a lightweight computer vision system for reconstructing the surface characteristics of a small object using four images captured under different lighting directions. The method is based on photometric stereo, which estimates surface normals and albedo from illumination changes while keeping the camera viewpoint fixed. The estimated normals are then integrated to obtain an approximate depth map. The project uses only classical image processing and linear algebra, making it suitable for low-resource academic environments. It avoids deep learning, large datasets, and GPU dependency while still producing visually meaningful outputs such as object mask, albedo image, normal map, depth map, and relit preview.

1. Introduction

Computer vision aims to extract meaningful information from images. While many student projects focus on recognition and detection, another important area is shape recovery from physical image cues. Photometric stereo is a classical technique that estimates the orientation of a surface by observing it under varying illumination. This project applies that idea in a simple and low-cost manner using only a mobile phone camera and controlled lighting.

2. Problem Statement

Most common student computer vision projects are repetitive and may depend on pretrained models or larger hardware resources. The goal of this work is to build a more unique, syllabus-driven, and computationally efficient project that demonstrates real understanding of computer vision principles.

3. Objectives

  • Build a low-resource project from the course syllabus.
  • Estimate albedo and surface normals from multiple light directions.
  • Recover an approximate depth map from the normal field.
  • Generate clear and interpretable outputs for demonstration.

4. Syllabus Mapping

  • Module 1: image formation, filtering, intensity normalization
  • Module 3: segmentation and object extraction
  • Module 5: albedo estimation, photometric stereo, Shape From X

5. Methodology

  1. Capture four aligned images of the object under left, right, top, and bottom lighting.
  2. Convert the images to grayscale and apply Gaussian denoising.
  3. Normalize image intensity values.
  4. Generate an object mask using thresholding and morphology.
  5. Solve the photometric stereo equation I = Lg using known light directions.
  6. Compute albedo and surface normals.
  7. Integrate the normal field using an FFT-based method to estimate depth.
  8. Save all outputs for visualization and analysis.

6. Software and Hardware Requirements

  • Python 3.10 or above
  • NumPy
  • OpenCV
  • Mobile phone camera
  • Basic directional light source such as a phone flashlight

7. Expected Results

The system produces the following outputs:

  • binary object mask
  • albedo image
  • surface normal map
  • approximate depth map
  • relit preview image

8. Advantages

  • low resource usage
  • no deep learning required
  • strong syllabus relevance
  • more unique than common classroom projects
  • easy to explain mathematically

9. Limitations

  • best suited for matte objects
  • requires fixed camera and object position
  • depth is approximate rather than exact
  • sensitive to harsh shadows and reflective surfaces

10. Conclusion

The project demonstrates that meaningful surface reconstruction can be achieved using classical computer vision with very limited resources. It combines theoretical relevance, uniqueness, simplicity of implementation, and good presentation value, making it a strong academic project for a Computer Vision course.
