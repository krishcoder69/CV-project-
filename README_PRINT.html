Low-Cost 3D Surface Reconstruction Using Photometric Stereo

Overview

This repository contains a lightweight computer vision project that reconstructs the surface shape of a small object using only a mobile phone camera and four images captured under different lighting directions. The project is based on photometric stereo, a classical computer vision technique from the Shape From X portion of the syllabus.

Problem Statement

Many student computer vision projects focus on common topics such as face detection, attendance systems, object detection, and tracking. These are often repetitive and may require pretrained models or larger computational resources. This project offers a more original and academically grounded solution that demonstrates real computer vision concepts while keeping resource usage low.

Proposed Solution

The system estimates the surface characteristics of an object by keeping the camera fixed and changing the light direction across four input images. It computes an object mask, albedo map, normal map, approximate depth map, and a relit preview image.

Why This Project Is Unique

  • It is based on photometric stereo, which is uncommon in student submissions.
  • It uses classical vision theory instead of a pretrained deep learning model.
  • It produces visually strong results with low computation.
  • It directly reflects advanced syllabus topics.

Syllabus Relevance

  • Module 1: image formation, filtering, image enhancement, intensity normalization
  • Module 3: segmentation and object extraction
  • Module 5: light at surfaces, albedo estimation, photometric stereo, Shape From X

Implementation Summary

  1. Read four images of the same object under left, right, top, and bottom illumination.
  2. Convert images to grayscale and apply denoising.
  3. Normalize intensity.
  4. Segment the object from the background.
  5. Solve the photometric stereo equations to estimate albedo and normals.
  6. Integrate the normal field to obtain an approximate depth map.
  7. Save the visual outputs.
